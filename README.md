# BofaDueDiligence

Plateforme de conformité Anti-Money Laundering (AML) et Enhanced Due Diligence (EDD) — gestion des dossiers, vérification documentaire et suivi réglementaire.

## Technologies

- **Backend :** PHP 7.4+
- **Base de données :** MySQL 8.0+
- **Serveur web :** Apache avec `mod_rewrite`
